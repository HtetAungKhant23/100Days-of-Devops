commands
========

- !! -> can used to re-execute the last command

- useradd -> create a new user account

- chage -> short term of "change age" -> Manages account aging parameters

- hostnamectl -> check os

- cat /etc/os-release -> check os detail

- which -> check location

- chown -> change ownership
eg: chown -R newuser:newgroup /path/to/directory
    -R -> recursive
    -v -> verbose output
